# KugouLayout
an interesting layout

一个模仿酷狗播放器滑动返回的layout

可以让你的app有更丰富 便捷的手势操作

支持activity滑动返回和普通layout的滑动 显示/隐藏 两种模式,可以参考demo里面的Activity1和Acitivyt2

# 个人建议使用场景

用于activity的滑动返回

需要使用网络加载要长时间等待的activity,可以用KugouLayout加入摇屏功能,让用户可以在等待加载期间拽一拽屏幕
